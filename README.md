# 🔐 WatchWave — AI-Driven Wi-Fi Intrusion Monitor

> BSc (Hons) Cyber Security — Final Year Project | University of West London (RAK)

## 📌 Overview
WatchWave is an AI-powered intrusion detection system that monitors Wi-Fi 
network traffic in real time, detecting anomalous behaviour using machine 
learning to flag potential threats before they escalate.

## 🚀 Key Features
- 📡 Real-time Wi-Fi packet capture and traffic analysis
- 🤖 ML-based anomaly detection and threat classification
- 🚨 Alert system for suspicious network activity
- 📊 Traffic visualisation dashboard
- 🖥 Built and tested on Kali Linux

## ⚙️ Setup & Installation
```bash
# Clone the repository
git clone https://github.com/mazheranvar/watchwave.git
cd watchwave

# Install dependencies
pip install -r requirements.txt

# Run WatchWave
python watchwave.py
```
